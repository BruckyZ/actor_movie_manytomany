About the com.example.actor_movie_manytomany.entities.Movie database

This application was developed for the java bootcamp Oct 2017.\It has the folowing features:

  1. Many to many relation between movies and actors
  2. A form to add actors to the database
  3. A form to add movies to the database
  4. A form to add actors to movies
  5. thymeleaf fragments, inlcuding:
    • A condition to display links only when actors have been entered into the database and there are still some actors to be added to the movies
    • A navigation bar that indicates where in the application a user is
  6. a method has been added to the cloudinary configuration to transform images based on the fetured actors face. H ave a look at this code as well
    7. You will be given the code of this application to type out. Make sure you dont leave anything out!
